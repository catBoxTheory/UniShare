# UniShare 雙語字幕擴展

這個 Chrome 擴展可以在 YouTube 影片上顯示即時雙語字幕（英文 + 繁體中文）。

## 安裝步驟

1. 打開 Chrome 瀏覽器
2. 進入 `chrome://extensions/`
3. 開啟右上角的 **開發者模式**
4. 點擊 **載入未封裝項目**
5. 選擇 `extension` 資料夾
6. 完成！

## 使用方法

1. 前往 YouTube 觀看任何有英文字幕的影片
2. 確保影片的字幕已開啟（點擊 CC 按鈕）
3. 擴展會自動偵測字幕並翻譯成繁體中文
4. 點擊擴展圖標可以調整設定

## 功能

- ✅ 即時翻譯英文字幕為繁體中文
- ✅ 攔截 YouTube 的字幕數據
- ✅ 支援自動生成的字幕
- ✅ 可切換顯示英文/中文
- ✅ 翻譯結果快取，避免重複翻譯

## 技術說明

- 使用 DeepSeek API 進行翻譯
- 攔截 YouTube 的 timedtext API 請求
- 觀察 DOM 變化來同步字幕

## 注意事項

- 需要影片本身有英文字幕（手動或自動生成）
- 翻譯速度取決於網路連線
- API Key 已內建，無需額外設定
